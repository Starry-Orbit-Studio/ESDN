## 如何查看游戏帧率

进入游戏后，屏幕左下角即为实时帧率显示器

![](image/README/1701940390281.webp)

`FPS`: 当前游戏渲染帧率
`AVG`: 当前对局平均帧率
`Detail`: 游戏画面细节等级
